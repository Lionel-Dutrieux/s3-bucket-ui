# Security Policy

## Reporting a vulnerability

Please **do not open a public issue** for security problems.

Report vulnerabilities privately through
[GitHub Security Advisories](https://github.com/Lionel-Dutrieux/s3-bucket-ui/security/advisories/new).
You'll get a response as soon as possible, and a fix will be released before
any public disclosure.

## Supported versions

Only the latest release (and the `latest` Docker tag) receives security
fixes.
